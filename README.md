# Sales Forecasting Project

This project predicts future sales using Machine Learning in Python.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Project Description
The project uses Linear Regression to predict future sales based on previous month data.

## Features
- Sales prediction
- Future forecasting
- Graph visualization

## Output Example
Month 11: 263.33
Month 12: 279.58
Month 13: 295.82
